# Energy System Analysis
A time‑series analysis project exploring national energy system behaviour, including demand patterns, renewable generation trends, and rolling‑window modelling.


## Project Overview
This project analyses large‑scale energy system data to understand how demand and renewable generation evolve over time.
The focus is on identifying trends, anomalies, and system‑level behaviour using rolling windows and combined renewable metrics.


## Objectives
- Analyse daily demand and renewable generation

- Build rolling‑window trend models

- Combine wind + solar into a unified VRE metric

- Identify peaks, troughs, and anomalies

- Produce clear visual insights for interpretation


## Key Features
- Time‑series cleaning and preprocessing

- Rolling averages for smoothing

- Combined Wind+Solar generation metric

- Trend visualisation

- Anomaly identification

- Clear, reproducible notebook workflow


## Tools & Technologies
- Python

- Pandas

- NumPy

- Matplotlib / Seaborn

- Jupyter Notebook


## How to Use
- Clone or download the repository

- Open the Jupyter notebook

- Run all cells to reproduce the analysis

- View generated graphs in the notebook
